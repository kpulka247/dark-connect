Dark Connect

How to Use

Dark Connect works automatically! Simply install the extension, and it will instantly apply a dark theme to:

  • connect.garmin.com
  • livetrack.garmin.com
  • live.garmin.com

No configuration is needed. Enjoy a more comfortable browsing experience!
